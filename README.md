# Swiggy Business Analysis Dashboard

## Project Overview
This project is an interactive Swiggy Business Analysis Dashboard created using Microsoft Excel.

The dashboard analyzes:
- Total Sales
- Average Ratings
- Average Order Value
- Total Orders
- Sales Trends
- State-wise Performance
- Food Category Analysis

- Microsoft Excel

## Key Insights
1. Bengaluru generated the highest revenue.
2. Veg category contributed a major share in sales.
3. Weekend sales are higher than weekdays.
4. Customer ratings remained consistently positive.
5. Interactive dashboard helps in quick business decisions.

## Business Use Case
This dashboard helps stakeholders:
- Monitor sales performance
- Identify top-performing cities
- Analyze customer satisfaction
- Track order trends
- Make data-driven decisions
